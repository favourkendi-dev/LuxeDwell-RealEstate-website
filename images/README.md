### Project Title
LUXEDWELL REAL ESTATE WEBSITE
### Project Description
LuxeDwell is a high end  responsive web platform designed for Nairobi’s premier real estate market. The project focuses on bridging the gap between architectural art and functional urban living, providing a bespoke digital experience for high-net-worth investors and discerning homeowners in prime nodes like Karen, Westlands and Muthaiga.

### Installation
1. Clone the Repository :
``` bash
git clone  git@github.com:favourkendi-dev/luxedwell-website

```

2. Install Independencies
``` bash
npm install

```

### Usage
- To run the project ,use the following commands
``` bash
npm start

```
